# نظام أرشفة الوثائق - Document Archiving System

## وصف النظام
نظام ويب كامل باللغة العربية لأرشفة الوثائق في المكاتب. يدعم رفع الملفات، البحث، التقارير، رموز QR، والنسخ الاحتياطي.

## المستخدمين
- admin / admin
- user / user

## كيفية التشغيل
1. في مجلد المشروع:
   ```
   pip install -r requirements.txt
   ```

2. تشغيل الخادم:
   ```
   python run.py
   ```

3. الوصول من المتصفح:
   - localhost:5000
   - أو عنوان IP المحلي:5000 من الجهاز الثاني في الشبكة نفسها

## الميزات
- تسجيل دخول آمن
- إضافة/تعديل/حذف الوثائق (وارد/صادر)
- رفع ملفات متعددة (PDF, صور)
- معاينة وتحميل الملفات
- بحث وفلترة متقدم
- تقارير يومية/شهرية قابلة للطباعة
- رموز QR فريدة لكل وثيقة
- نسخ احتياطي يدوي لقاعدة البيانات والملفات

## هيكل المشروع
```
.
├── app.py          # التطبيق الرئيسي
├── models.py       # نماذج قاعدة البيانات
├── config.py       # الإعدادات
├── run.py          # تشغيل التطبيق
├── requirements.txt
├── templates/      # صفحات HTML
├── static/         # CSS, JS, uploads/
├── database.db     # قاعدة بيانات SQLite (تُنشأ تلقائياً)
└── backups/        # النسخ الاحتياطية
```

## النسخ الاحتياطي التلقائي
- زر النسخ في القائمة
- ينشئ ZIP لقاعدة البيانات + الملفات

## ملاحظات
- يعمل بدون إنترنت (ملفات محلية)
- قاعدة بيانات SQLite في نفس المجلد
- رفع ملفات حتى 16MB
- الوصول من الشبكة المحلية (0.0.0.0:5000)
